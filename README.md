# 🎵 SpotifyCraft — Spotify-подібний музичний плеєр для Minecraft

**Версія:** 1.0.0  
**Minecraft:** 1.21.1  
**Mod loader:** Fabric  

---

## ✨ Що вміє мод

- 🎨 Красиве меню у стилі Spotify (темна тема, зелений акцент)
- 📁 Автоматично створює папку `.minecraft/spotifycraft_music/`
- 🎵 Програє `.wav`, `.au`, `.aiff` файли з папки
- ⏮ ⏯ ⏭ Кнопки: попередній / пауза / наступний трек
- 🔀 Shuffle (випадковий порядок)
- 🔁 Loop (повтор треку)
- 🔊 Регулятор гучності
- ⏩ Перемотування треку (клікай на смугу прогресу)
- 📜 Список треків з прокруткою
- ⌨️ Клавіатурні скорочення

---

## 🎮 Клавіші

| Клавіша | Дія |
|---------|-----|
| **M** | Відкрити/закрити плеєр |
| **Пробіл** | Пауза / Програти |
| **→** | Наступний трек |
| **←** | Попередній трек |
| **ESC** | Закрити плеєр |

## 🚀 ЯК ЗІБРАТИ МОД (без програмування, БЕЗКОШТОВНО)

### Крок 1 — Завантаж GitHub Desktop
1. Іди на https://desktop.github.com
2. Скачай і встанови GitHub Desktop
3. Зареєструйся або увійди в акаунт GitHub

### Крок 2 — Створи репозиторій
1. Відкрий GitHub Desktop
2. Натисни **"Create a New Repository on your hard drive"**
3. Назва: `spotifycraft`
4. Натисни **Create Repository**

### Крок 3 — Скопіюй файли моду
1. GitHub Desktop покаже папку репозиторію (натисни "Show in Explorer")
2. Скопіюй **всі файли з архіву** (що ти завантажив) у цю папку

### Крок 4 — Завантаж на GitHub
1. У GitHub Desktop ти побачиш всі нові файли
2. Внизу зліва напиши будь-що у полі "Summary", наприклад: `Initial commit`
3. Натисни **Commit to main**
4. Натисни **Publish repository** (або Push origin)
5. Переконайся що **"Keep this code private"** — на твій вибір
6. Натисни **Publish Repository**

### Крок 5 — GitHub збере мод автоматично!
1. Відкрий браузер і іди на github.com → твій акаунт → репозиторій `spotifycraft`
2. Натисни вкладку **Actions** (зверху)
3. Ти побачиш workflow **"Build SpotifyCraft Mod"** — він вже запускається!
4. Зачекай ~3-5 хвилин (зелена галочка = успіх ✅)
5. Натисни на workflow → **Artifacts** → скачай **SpotifyCraft-mod.zip**
6. Розпакуй — всередині буде `spotifycraft-1.0.0.jar` — це твій мод!

## 📦 Встановлення моду

### 1. Встанови Fabric Loader
- Іди на https://fabricmc.net/use/
- Скачай Fabric Installer
- Запусти, обери версію **1.21.1**, натисни Install

### 2. Встанови Fabric API
- Іди на https://modrinth.com/mod/fabric-api
- Скачай для версії **1.21.1**
- Поклади в `.minecraft/mods/`

### 3. Встанови SpotifyCraft
- Поклади `spotifycraft-1.0.0.jar` в папку `.minecraft/mods/`

### 4. Запусти Minecraft
- Вибери профіль **Fabric 1.21.1** у лаунчері
- Натисни Play!

## 🎵 Додавання музики

1. Запусти Minecraft з модом хоча б раз — мод створить папку автоматично
2. Знайди папку: `.minecraft/spotifycraft_music/`
   - Windows: `C:\Users\ТвоєІм'я\AppData\Roaming\.minecraft\spotifycraft_music\`
   - Або відкрий Minecraft → Options → Resource Packs → Open Pack Folder → піди на рівень вище
3. Поклади туди музичні файли формату **.wav**
4. У грі натисни **M**, потім **"⟳ Refresh"** у лівому меню

### Конвертація MP3 → WAV (безкоштовно)
- Онлайн: https://cloudconvert.com/mp3-to-wav
- Або: https://convertio.co/mp3-wav/

## ❓ Часті питання

**Q: Мод не з'являється в грі?**  
A: Переконайся що встановлено Fabric API і правильна версія Minecraft (1.21.1)

**Q: Музика не грає?**  
A: Переконайся що файли у форматі .wav, не .mp3

**Q: Де папка .minecraft?**  
A: Windows: Win+R → `%appdata%\.minecraft`

**Q: Можна використовувати з іншими модами?**  
A: Так! Мод тільки додає плеєр, не змінює гру.
